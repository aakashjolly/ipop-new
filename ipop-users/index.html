IPOP Users

IPOP has been used in various cloud computing research projects, including:

  • The Grid Appliance in the Archer project uses IPOP to create virtual private clusters for running batches of long-running simulations across distributed virtual machines running Linux and the open-source HTCondor job scheduler.
  • The FutureGrid testbed includes a distributed infrastructure for research on cloud middleware and applications, and uses IPOP to allow users to create their own private virtual networks for education/training.
  • The ConPaaS project uses IPOP to enable the deployment of various Platform-as-a-Service (PaaS) systems across multiple cloud providers.
  • The PRAGMA project uses IPOP to create virtual network overlays for collaborative environments in e-Science applications.
